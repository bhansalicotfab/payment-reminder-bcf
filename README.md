## How to Use

1. **Open the app** from your home screen
2. **Tap "Sync from Drive"** to load your ledger data
3. **Search** for any party name
4. **Filter** by voucher type (Sales, Purchase, etc.)
5. **View balances** - Green = receivable, Red = payable

## Data Privacy

- All data is stored locally on your phone
- No data is sent to any server
- Only syncs with your Google Drive when you tap "Sync"
